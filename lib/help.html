Vague - Velvet Assembler Graphical User Environment

Help

Vague is a frontend to the Velvet assembler.

Usage

  1. Specify an output directory
  2. Specify one or more read files - select if these are single or paired end reads.
  3. Optional - Use the "Estimate best k-mer" button to estimate a good k-mer size to use
  4. Hit Run! and wait for your assembly to complete!

Tabs

  • The "Main" tab allows configuration of the common options.
  • The "Advanced" tab allows setting of the more advanced configuration for velvet
  • The "Log" tab shows the output of the running velvet process
  • The "Results" tab shows the assembly output once velvet has finished running
Good luck!
