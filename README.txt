Vague - Velvet Assembler Graphical User Environment

Vague is a frontend to the Velvet assembler, designed to make it easy
to use velvet for the most common options.

Source available at https://github.com/Victorian-Bioinformatics-Consortium/vague

Written by David R. Powell (david.powell@monash.edu)

=== DEPENDENCIES ====

 - velvet >= 1.2.07
 - jre >= 1.6
 - velvetk >= 1.0 (optional)
